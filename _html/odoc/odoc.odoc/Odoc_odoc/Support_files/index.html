Sourceval write : ?without_theme:bool -> Fs.Directory.t -> unit

write ?without_theme output_dir copies the support files to the output_dir. If without_theme is true the theme will not be copied, the default value is false.

Sourceval print_filenames : ?without_theme:bool -> Fs.Directory.t -> unit

Prints, to STDOUT, the names of the files that calling Support_files.write would output, one filename per line.
