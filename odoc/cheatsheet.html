Code Blocks

Inline [code].

{[
  let _ = "Block code"
]}

{foo@text[
  Code block with {[inner code block syntax]}
]foo}

{@python[
  [i+1 for i in xrange(2)]
]}

Inline code.

let _ = "Block code"
Code block with {[inner code block syntax]}
[i+1 for i in xrange(2)]
